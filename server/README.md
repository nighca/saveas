Server of SaveAs
======

This is server for SaveAs 

(A chrome extension to save resource to [Qiniu Cloud Storage](http://qiniutek.com/ "Qiniu"))

 * Author:   nighca

 * Mail:     nighca@live.cn
 
结构
----

 * 服务端
 
   node.js + express + mysql

依赖
----

 * 需安装node.js

运行方法
----
   
   / 目录下

   $ npm install
   
   $ node app.js
